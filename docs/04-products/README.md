This section stores product-specific strategy, roadmap, positioning, and operating guidance for commercial products built on Clada OS.

It does not replace [../01-product](../01-product/README.md), which defines the platform product model and module architecture. It also does not replace [../04-features](../04-features/README.md), which stores feature specifications.

## Purpose

Product strategy documents define why a product exists, who it serves, what outcomes it should create, and how it should evolve over multiple maturity stages.

Feature specifications, sprint plans, release specifications, and implementation work should align to the relevant product strategy document before changing product behaviour.
