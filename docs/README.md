# Clada OS Documentation

This folder is the source of truth for Clada OS product, design, engineering, feature, decision, and sprint planning.

## Read order for Codex

Before making changes, read these documents in order:

1. `docs/README.md`
2. The active sprint in `docs/06-sprints/`
3. Relevant company, product, design, engineering, feature, and decision documents

## Folder structure

- `00-company/` company direction
- `01-product/` product strategy
- `02-design/` design standards
- `03-engineering/` engineering standards
- `04-features/` feature specifications
- `05-decisions/` architecture decision records
- `06-sprints/` sprint plans
- `07-research/` market and customer research
- `99-archive/` superseded documents

## Rule

If a feature conflicts with these docs, update the relevant document first or do not build the feature.
